# Logo Files

Place your logo files in this directory.

## Recommended Files

- **logo.png** - Main logo (recommended size: 200x60px or similar aspect ratio)
- **logo.svg** - Vector version (optional, better for scaling)
- **favicon.ico** - Browser favicon (16x16 or 32x32px)

## File Naming

The template is configured to look for:
- `logo.png` - Main logo used in navigation

## Image Guidelines

- **Format**: PNG with transparent background (recommended) or SVG
- **Size**: Recommended width 150-200px, height auto (maintain aspect ratio)
- **Colors**: Should work well on orange background (#ea580c / orange-600)
- **Style**: Clean, professional, representing African marketplace theme

## Usage

The logo will automatically appear in:
- Navigation bar (top left)
- All pages (via base template)

## Adding Your Logo

1. Place your logo file as `logo.png` in this directory
2. The template will automatically use it
3. If the file doesn't exist, it will fall back to text-only branding

## Example

```
static/
  logo/
    logo.png    <- Your main logo here
    logo.svg    <- Optional vector version
    favicon.ico <- Optional favicon
```
